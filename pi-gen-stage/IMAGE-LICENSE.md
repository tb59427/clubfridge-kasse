## Raspberry Pi OS (Debian Trixie)

Dieses Image basiert auf **Raspberry Pi OS Lite** (Debian Trixie, arm64),
erstellt mit [pi-gen](https://github.com/RPi-Distro/pi-gen).

Raspberry Pi OS enthält Software unter verschiedenen Open-Source-Lizenzen,
darunter GNU GPL, LGPL, BSD, MIT und Apache 2.0. Die vollständigen
Lizenztexte sind auf dem Image unter `/usr/share/doc/*/copyright` einsehbar.

Der Quellcode aller GPL/LGPL-lizenzierten Pakete ist über die offiziellen
Paketquellen verfügbar:

- Debian: https://sources.debian.org
- Raspberry Pi: https://archive.raspberrypi.com/debian/

Alternativ kann der Quellcode auf schriftliche Anfrage an info@clubfridge.com
bereitgestellt werden (gültig für 3 Jahre ab Veröffentlichung des Images).
